# NaiviSense вЂ” AI Powered Therapy Platform

A Flutter mobile application connecting **Admins, Therapists, and Parents** to coordinate therapy for children with developmental needs. Built on clean, feature-based architecture with Riverpod for state and GoRouter for navigation.

## рџЋЇ Roles

- **Admin** вЂ” Add therapists, add children (multi-step form), assign therapists to children
- **Therapist** вЂ” Today's sessions dashboard, child profiles with 5 tabs (Overview, Schedule, Tasks, Diet, Feedback), session notes with ratings, weekly feedback, progress reports with charts
- **Parent** вЂ” Today's plan, tasks, daily feedback, urgent issue reporting, progress reports

### Principles

- **Riverpod** for all state (`StateNotifierProvider`, `Provider.family` for per-child data)
- **GoRouter** for declarative navigation
- **Mock repository** injected via `repositoryProvider` вЂ” real API swap is a single file
- **Models** are immutable with `copyWith`
- **Design system** lives in `core/theme` and `shared/widgets` вЂ” used everywhere, never hardcoded

## рџ”Њ Swapping the mock repository for a real API

All data flows through `MockRepository` (`lib/data/mock/mock_repository.dart`). To connect a real backend:

1. Create `lib/data/remote/api_repository.dart` with the same method signatures
2. Change `repositoryProvider` in `lib/features/auth/providers/auth_provider.dart` to return the API impl
3. Done вЂ” no UI code changes needed
